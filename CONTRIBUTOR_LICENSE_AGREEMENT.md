# 蓝域星河有限公司集团 开源项目通用贡献者许可协议（CLA）

**版本**：1.0.0
**最后更新日期**：2027-02-28
**适用范围**：蓝域星河有限公司集团旗下所有开源子项目（“星海量子”生态）


感谢您对 **蓝域星河有限公司集团**（以下简称“版权方”或“我们”）旗下开源项目的关注与贡献。

在向版权方旗下任何开源项目提交任何代码、文档、设计或其他材料（以下简称“贡献”）之前，请您仔细阅读并同意本通用贡献者许可协议（以下简称“本协议”）的全部条款。


## 一、定义

1.1 **“版权方”** 指蓝域星河有限公司集团，即本协议覆盖的所有开源项目的合法所有者及版权持有者。

1.2 **“贡献者”** 指向版权方旗下任一开源项目提交贡献的自然人或实体。

1.3 **“贡献”** 指贡献者提交给版权方旗下任一开源项目的任何代码、补丁、文档、设计、数据或其他材料，包括但不限于通过 GitHub Pull Request、Issue 附件、邮件或其他任何形式提交的内容。

1.4 **“项目”** 指版权方旗下以“星海量子”、“StarseaOS”或相关品牌发布的所有开源子项目，包括但不限于：

   - WebApp 加载器（StarseaOS WebApp Loader）
   - 后续发布的所有开源模块、工具、库、驱动程序及文档

   （以下简称“项目”或“各项目”）

1.5 **“集团”** 指蓝域星河有限公司集团及其关联公司。


## 二、版权授予

2.1 贡献者特此授予版权方及其后继者一项 **永久性、全球性、免版税、不可撤销、非排他性** 的许可，允许版权方：

   a) 以任何方式使用、复制、修改、分发、合并、出版、展示、表演、传播贡献的全部或部分内容；

   b) 将贡献纳入项目及其衍生作品中；

   c) 将贡献用于 **开源版本**（包括但不限于 AGPLv3、MIT、Apache 等开源许可证）和 **商业闭源版本**（包括但不限于版权方商业产品的专业版、企业版订阅）中；

   d) 将贡献以 **任何许可证形式**（包括但不限于开源许可证、商业专有许可证）进行再许可或分发，无需另行征得贡献者的同意或向贡献者支付任何费用。

2.2 贡献者确认，本协议授予的权利包括但不限于版权、专利权及相关知识产权。


## 三、贡献者声明与保证

3.1 贡献者声明并保证：

   a) 贡献是贡献者的 **原创作品**，或贡献者拥有足够的权利授予本协议中所述的许可；

   b) 贡献 **不侵犯** 任何第三方的知识产权、商业秘密或其他合法权利；

   c) 贡献 **不包含** 任何恶意代码、病毒、后门、时间炸弹或其他有害程序；

   d) 如贡献包含第三方的代码或材料，贡献者已获得该第三方的 **合法书面授权**，并有能力将其按本协议授予版权方；

   e) 贡献者 **未** 在贡献上设置任何可能限制版权方行使本协议授予权利的技术保护措施。

3.2 如贡献是贡献者在受雇期间创作的，贡献者声明其 **雇主已书面放弃** 对贡献的所有权利，或贡献者已获得雇主书面授权进行本次贡献。


## 四、专利授予

4.1 贡献者特此授予版权方及其后继者一项 **永久性、全球性、免版税、不可撤销、非排他性** 的专利许可，允许版权方制作、使用、销售、许诺销售、进口、分发贡献及包含贡献的衍生作品。

4.2 上述专利许可仅限于贡献者 **现有及未来** 因贡献而产生的必要专利权利要求。


## 五、贡献者的权利保留

5.1 贡献者保留对贡献的所有权及知识产权，但根据本协议授予版权方的权利除外。

5.2 贡献者可以自由地将贡献用于其他项目，但不得阻止版权方行使本协议授予的权利。


## 六、免责声明

6.1 贡献按“现状”提供，不作任何明示或默示的保证，包括但不限于适销性、特定用途适用性及非侵权保证。

6.2 贡献者不对因使用或分发贡献而产生的任何直接或间接损失承担责任，即使已被告知此类损害的可能性。


## 七、协议效力

7.1 本协议自贡献者向版权方旗下任一开源项目提交贡献之日起生效，并对贡献者及其继承人、受让人具有约束力。

7.2 贡献者理解并同意，本协议 **不可撤销**，版权方可 **永久依赖** 本协议使用贡献。

7.3 贡献者同意，版权方可随时修改本协议，修改后的协议将在版权方官方网站或项目仓库中公布后生效。贡献者在此后继续提交贡献即视为接受修改后的条款。


## 八、适用法律与争议解决

8.1 本协议受 **中华人民共和国法律** 管辖，并按其解释。

8.2 因本协议引起的或与本协议有关的任何争议，双方应首先通过友好协商解决。协商不成的，任何一方均有权向 **版权方所在地有管辖权的人民法院** 提起诉讼。


## 九、确认与签署

**通过向版权方旗下任一开源项目提交 Pull Request、Issue 附件或任何形式的贡献，即视为贡献者已阅读、理解并同意本协议的全部条款。**

如贡献者代表实体（如公司、组织）进行贡献，声明该实体已正式授权贡献者代表其接受本协议。
